# 大学生智能职业规划系统

一个面向大学生的智能职业规划 Web 系统，旨在结合用户基本信息、能力评估、发展路径分析与职业推荐，帮助用户更清晰地认识自己、规划成长路径，并获得个性化职业发展建议。

---

## 项目简介

本项目采用前后端分离架构开发：

- **前端**：Vue 3 + Vite + Vue Router + Pinia
- **后端**：FastAPI + SQLAlchemy
- **数据库**：MySQL
- **其他功能**：头像上传、静态资源访问、RESTful API 接口

当前项目已完成基础环境搭建，并实现了**用户注册 / 登录**、**个人信息管理**等核心基础功能，正在继续扩展能力评估、发展路径推荐、职业推荐与 AI 规划等模块。

---

## 项目目标

本系统希望实现以下功能：

- 用户注册与登录
- 个人信息填写与维护
- 成绩与能力分析
- 发展路径推荐（就业 / 考研 / 考公 / 出国）
- 职业推荐与职业详情展示
- AI 生成成长规划与职业建议
- 历史记录保存与可视化展示
